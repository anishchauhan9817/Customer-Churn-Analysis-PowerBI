# Customer Churn Analysis - Power BI

## 📊 Project Overview
This project analyzes telecom customer churn using Power BI.  
The goal is to identify patterns that cause customers to leave and help businesses improve retention.

## 🛠 Tools Used
- Power BI
- Excel / CSV
- Data Visualization
- DAX

## 📂 Dataset
The dataset contains customer information such as:
- Customer ID
- Gender
- Contract Type
- Monthly Charges
- Tenure
- Churn Status

## 📈 Dashboard Insights
- Total Customers
- Churn Rate
- Churn by Contract Type
- Churn by Payment Method
- Churn by Tenure
